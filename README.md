# Sistema de Cadastro e Agenda de Funcionários em C

Trabalho prático da disciplina de Introdução à Ciência da Computação.

## Descrição

Programa em C para cadastrar e gerenciar até 10 funcionários, usando matrizes e vetores paralelos.

## Funcionalidades

O sistema possui um menu com as opções:

1. Cadastrar novo funcionário
2. Mostrar todos os dados
3. Buscar funcionário por nome
4. Mostrar maior salário
5. Mostrar média salarial
6. Sair

## Como compilar

```bash
gcc main.c -o sistema_funcionarios
```

## Como executar

```bash
./sistema_funcionarios
```

## Critérios atendidos

* Uso de matrizes e vetores
* Controle de até 10 funcionários
* Menu interativo com repetição
* Funções recebendo matrizes e vetores como parâmetros
* Validação para operações sem funcionários cadastrados
* Cadastro, listagem, busca, maior salário e média salarial
